cucumber-example
================
non rails apps BDD Test example.

Init
---------------
### capybara-webkit install
https://github.com/thoughtbot/capybara-webkit/blob/v0.6.1/README.md

### gems install:
    gem install bundler
    bundle

Run
--------------
### default(webkit/x)
    cucumber

### webkit/headless
    cucumber headless=y

### selenium/firefox
    cucumber -p firefox
